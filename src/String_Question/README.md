### [005--替换空格](Solution005.java)
&nbsp;&nbsp;此题输入一个带空格的字符串，并将空格替换为“%20”。<br>
&nbsp;&nbsp;解题思路先进行一次遍历统计空格数，在设置替换空格后的新字符串长度为旧字符串长度+空格数x2，因为要替换的东西比空格长2。再设置两个指针，一个指向旧字符串的旧长度末尾，一个指向新字符串的新长度末尾，同时向前移动，遇到空格替换。<br>
