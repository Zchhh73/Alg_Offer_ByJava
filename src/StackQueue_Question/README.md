解题思路
------
栈是一种常见的数据结构，它的特点是后进先出，操作系统中系统会给每个线程创建一个栈存储函数调用时的参数，返回地址和临时变量等。在Java中栈使用Stack类实现，Stack继承自Vector


队列也是一种重要的数据结构，它的特点是先进先出，即第一个进入队列的元素将会第一个出来
